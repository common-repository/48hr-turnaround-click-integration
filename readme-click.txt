== Description ==

Accept digital payments on your website, smartphone apps or over the phone with Click by Paymark, our eCommerce gateway.

Built on our 25 years of experience in processing payments and running secure finance infrastructure, Click is our best-of-breed secure payment channel, giving online businesses the ability to take payment for goods and services with total confidence.


== Installation ==
	
1) Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly. 

Then Activate the plugin.

2) To configure the plugin, click on WooCommerce | Settings from the left hand menu, then the top tab "Payment Gateways". You should see "Click" as an option in the list. 

3) You should see "Click" displayed across the top of the page as a link also. Click on this to configure the plugin, and you will see a list of options:

-Tick to "Enable Click".

-The "Title" option allows you to determine what your customers will see this payment option displayed as. You might choose for this to be something like "Payments by Click" or "Credit card"

-The "Description" box allows you to set the description for the payment method on the payment page.

-Test Mode allows you to put test transactions through using the Click demo account. Ensure you have Click demo credentials entered below.

-Click Username is the Username supplied by Click.

-Click Account ID is the Account Id supplied by Click.

-Click Password is also supplied by Click.
